# Zoo-anwendung
C# Code mit Datenbank 


This project is a **console-based Zoo Management application written in C**.  
It demonstrates basic programming concepts such as:

- structs
- arrays
- functions
- user input handling
- loops and conditional logic

The program simulates a simple animal database where users can enter, display, and manage animal records in a zoo.

---

## Features

The Zoo Application includes the following basic features:

- Add new animals
- Display existing animal information
- Store animal data using structs
- Console input and output
- Menu-driven user interface

This project focuses on practicing structured C programming and creating modular code that interacts with the user via the command line.

---

## Concepts Used

This project demonstrates several core C programming concepts:

- `struct` usage for organizing data  
- Arrays for storing multiple animal entries  
- Functions for modularization  
- Loops (`for`, `while`)  
- User interaction with `scanf` and `printf`  
- Conditional logic (`if / else`)

These fundamentals are essential for creating interactive console applications in C.

## How to Compile

To compile the program using **GCC**:

Linux / macOS:


gcc main.c -o zoo_app


Run the compiled program:


./zoo_app


Windows:


zoo_app.exe
